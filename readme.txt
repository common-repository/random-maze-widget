Widget that creates a maze.

== Description ==

The Random Maze widget generates and displays a maze with  the help of the PHP GD library. The size of the maze can be customized.

== Installation ==

1. Upload the "randommaze" folder to your plugins directory.
2. Activate the plugin.
3. Go to Appearance -> Widgets, and drag the widget to your desired container.
